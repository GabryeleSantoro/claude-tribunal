Tribunal convenes a panel of **five adversarial AI personas** and forces them to argue, cross-examine, and converge on a confidence-weighted verdict вҖ” then writes it to disk. The chat reply is a one-line confirmation. The heavy work lands in `docs/tribunal/`.

Built for decisions that deserve more than a single model's gut feeling.

<br>

Each role has a fixed adversarial function:

| Persona | Function |
|---|---|
| **Domain Expert** | Field authority. Facts over assumptions. Replaceable via `--persona`. |
| **Devil's Advocate** | Stress-tests every position. Exists to break weak arguments. |
| **Systems Thinker** | Second and third-order effects. What are you not seeing? |
| **Logician** | Internal consistency. Surfaces hidden premises and contradictions. |
| **Mediator** | Synthesizes tensions into a verdict the panel can live with. |

Votes are **conditional** вҖ” each persona can lean toward support or reject without full commitment. The weighted average decides.

## в–ё Flags

<kbd>--help</kbd> вҖ” Print usage and exit.

<br>

| Flag | Values | Effect |
|---|---|---|
| `--depth` | `full` \| `brief` | **full** (default): rich topic analysis, full opening paragraphs, prose deliberation. **brief**: tighter, same vote math. |
| `--brief` | вҖ” | Shorthand for `--depth brief`. |
| `--full-log` / `--verbose` | вҖ” | Write the complete deliberation (panel, arguments, cross-exam) to the output file. Default: compact verdict only. |
| `--multi-agent` | вҖ” | Spawn each persona as a real subagent. Same weighted verdict, higher fidelity, more latency. |
| `--persona "вҖҰ"` | string | Replace the Domain Expert slot with a named custom expert. |
| `--domain вҖҰ` | string | Domain hint вҖ” e.g. `ethical`, `technical`, `legal`. Shapes role labels. |
| `--export` | `md` \| `json` \| `adr` | Append an extra export block to the output file. Markdown verdict is always written. |
| `--min-confidence N` | `0`вҖ“`100` | Flag weak consensus: warn when weighted confidence falls below N. |

## в–ё Output Files

Every run (except `--help` or an empty topic) writes:

```
docs/tribunal/{YYYY-MM-DD}_{slug}.md
```

The **slug** is a sanitized form of your full invocation вҖ” flags and topic together. Re-running the same command on the same day appends `-2`, `-3`, and so on.

```
docs/tribunal/
в”ңв”Җв”Җ 2026-05-15_brief-postgresql-jsonb-vs-document-db.md
в”ңв”Җв”Җ 2026-05-15_export-adr-adopt-strict-typescript.md
в””в”Җв”Җ 2026-05-15_multi-agent-full-log-crdt-edge-layer.md
```

**Default file:** compact verdict вҖ” topic metadata, Final Verdict, Votes, Reasoning Trail, footer.  
**With `--full-log`:** adds Panel, Arguments Summary, Cross-Examination, and Deliberation sections.  
**Chat reply:** always a short confirmation only. The file is the record.

## в–ё Vote Math

Each persona returns a position, optional lean, and confidence score:

| Position | Lean | Vote value `vбөў` |
|---|---|---|
| support | вҖ” | `+1` |
| reject | вҖ” | `вҲ’1` |
| conditional | toward\_support | `+0.5` |
| conditional | neutral | `0` |
| conditional | toward\_reject | `вҲ’0.5` |

Weighted consensus:

```
vМ„ = ОЈ(vбөў В· cбөў) / ОЈcбөў      where cбөў = confidence вҲҲ [0, 100]
```

`--min-confidence N` flags verdicts where the panel's aggregate confidence is below threshold вҖ” weak consensus is surfaced, not suppressed.
